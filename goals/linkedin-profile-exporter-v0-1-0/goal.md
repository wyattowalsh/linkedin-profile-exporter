# LinkedIn Profile Exporter v0.1.0

Build `linkedin-profile-exporter` v0.1.0 as a complete, OpenSpec-driven product release: a WXT-based multi-browser extension suite plus bookmarklet that extracts accessible LinkedIn profile data from the current page, validates it through a project-owned canonical schema, and exports it to JSON, YAML, CSV/XLSX tabular output, XML, and compact LLM-context Markdown. The release also includes a modern settings UI, local-explicit privacy behavior, docs, design system, generated assets, CI/CD, pre-commit, Justfile, nested agent instructions, and source-managed web-store materials.

Use `facts.md` as the accepted shared understanding and `plan.md` as the approved execution plan.

Done means every accepted fact in `facts.md` is implemented or explicitly accounted for, the approved plan has been followed or updated through review, OpenSpec validation passes, the documented local/CI verification commands pass, and the repo contains no unrelated staged or hidden cleanup work.
