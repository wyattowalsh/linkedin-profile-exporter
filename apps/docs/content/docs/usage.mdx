# Usage

Open a LinkedIn profile page, run extraction from the extension popup or side panel, review the
captured profile summary, choose export formats, and download or copy local exports based on the
delivery setting. Review-before-export is the default workflow.

The Include all fields setting adds diagnostics for readiness state, section coverage, provenance,
confidence, and automation notes when those details are needed for troubleshooting.
