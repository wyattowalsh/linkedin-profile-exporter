# LinkedIn Profile Exporter

LinkedIn Profile Exporter extracts accessible LinkedIn profile data from the current page, validates it through the project-owned canonical schema, and exports local files for review and reuse. The canonical schema keeps LinkedIn-specific sections such as courses, featured items, organizations, interests, test scores, and patents when the page provides them.

The default product does not upload extracted data, send analytics, require LinkedIn credentials, or use a remote service. Clipboard delivery is available for text formats when users prefer copying over downloading.
