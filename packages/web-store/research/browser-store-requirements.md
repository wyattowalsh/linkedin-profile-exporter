# Browser Store Requirements

- Chrome and Edge use WXT ZIP outputs and source-managed listing metadata.
- Firefox requires add-on ZIP plus source ZIP review material.
- Safari uses WXT Safari output and the Xcode Safari Web Extension wrapper path.
- Mobile Safari and mobile Chrome are documented platform packaging and QA paths for v0.1.0.
- GitHub Releases provide the credential-free public artifact handoff for browser ZIPs, bookmarklet assets, Firefox source-review material, checksums, and release notes.
- Credentialed submission is out of scope.
